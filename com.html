# Com

EpiJS module for representing communities

Import it with:

       const com = require('@quantalabs/epijs').com

# Virus

Class representing a virus, which can infect a community.

# Parameters

  • rnaught Number (opens new window) The disease's R-Naught
  • u Number (opens new window) The disease's recovery rate
  • a Number (opens new window) The disease's incubation period (optional, default 0)
  • d Number (opens new window) The disease's death rate (for infected population) (optional, default 0)
  • h Number (opens new window) The disease's hospitalization rate
  • dh Number (opens new window) The disease's death rate (for hospitalized population)

# Examples

let covid = new Virus(5.7, 2.1/100)

# Community

Class representing a community, which can be infected with a disease, and compared to other communities.

# Parameters

  • pop Number (opens new window) The population of the community
  • i Number (opens new window) The start infected population of the community.
  • s Number (opens new window) The start susceptible population of the community

# Examples

let NewYorkCity = new Community(8419000, 300, 8418700)

# sir

SIR model for the community

# Parameters

  • disease Number (opens new window) A virus class. The virus to infect the community with and model for.
  • time Number (opens new window) Time to predict for.

# Examples

let NewYorkCity = new Community(8419000, 300, 8418700)
let covid = new Virus(5.7, 2.1/100)

outbreak = NewYorkCity.sir(covid, 100)

# seir

SEIR model for the community

# Parameters

  • disease Number (opens new window) A virus class. The virus to infect the community with and model for.
  • time Number (opens new window) Time to predict for.

# Examples

let NewYorkCity = new Community(8419000, 300, 8418700)
let covid = new Virus(5.7, 2.1/100, a=1/8)

outbreak = NewYorkCity.seir(covid, 100)

# seird

SEIRD model for the community

# Parameters

  • disease Number (opens new window) A virus class. The virus to infect the community with and model for.
  • time Number (opens new window) Time to predict for.

# Examples

let NewYorkCity = new Community(8419000, 300, 8418700)
let covid = new Virus(5.7, 2.1/100, a=1/8, d=1/100)

outbreak = NewYorkCity.seird(covid, 100)

# compare

Compare's two different outbreaks, communities, or anything else.

# Parameters

  • c HTMLCanvasElement (opens new window) The canvas element
  • model1 Function (opens new window) The first model to compare.
  • model2 Function (opens new window) The second model to compare.
  • m1name String (opens new window) The name of the first model
  • m2name String (opens new window) The name of the second model
  • days Number (opens new window) The total amount of days to compare for.

# Examples

let NewYorkCity = new Community(8419000, 300, 8418700)
let covid = new Virus(5.7, 2.1/100)
let covid_variant = new Virus(5, 4/100)

let chart = document.getElementById('model')

covid_outbreak = NewYorkCity.sir(covid, 100)
variant_outbreak = NewYorkCity.sir(covid_variant, 100)

compare(chart, covid_outbreak, variant_outbreak, "COVID-19", "COVID-19 Variant", 100) // We chose 100 as the amount of days, but it could be 50 or 25, not the lenght of the prediction that was in the model.
