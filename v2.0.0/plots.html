# Plot

Graphs plots from the pre module.

Import it with:

       const plots = require('@epispot/epijs').plots

# plot

Plots a output of a model from the pre module.

# Parameters

  • model Array (opens new window) The output from the pre module function.
  • canvas HTMLCanvasElement (opens new window) The canvas to show the output.
  • days Number (opens new window) The amount of days to plot
  • colors Array (opens new window) Custom colors for the graph, in the same order the compartments are. (optional, default null)
  • options Object (opens new window) Optional. Custom configuration to pass into the options parameter for chart.js, defaults to:
    {
     title: {
         display: true,
         text: "Total Cases"
     },
     scales: {
         yAxes: [{
             ticks: {
                 beginAtZero: true
             }
         }]
     }
    }

# Examples

let sirout1 = sir(4, 9999, 1000, 100, 1/21, 10999, true)

plot(sirout1, "canvas-pre1", 100) // Plots data for 100 days onto the canvas-pre1 chart, with the data from the SIR model.

Returns any Returns the chart.js chart, if needed for modification.
